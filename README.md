# Deally

Бот-миниигра для симуляции торговли криптовалютой.

# Запуск

Установи нужные библиотеки:
```bash
pip3 install aiogram
```

Установи `API_KEY` в `config.py`:
```python
# Config.py

API_KEY = '62843...fO70g'
```

Запусти бота (`run.py`):
```bash
cd DeallyBot
```
```bash
python3 run.py
```
